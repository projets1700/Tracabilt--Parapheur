Guide d'import dans Figma
Importer ce fichier dans Figma
1
Installer le plugin "HTML to Figma"
Rechercher dans Figma Community → Plugins → "HTML to Figma" by BuilderIO
2
Ouvrir ce fichier dans Chrome
Double-cliquer sur maquette_parapheurs.html ou glisser dans Chrome
3
Lancer le plugin dans Figma
Dans Figma → Plugins → HTML to Figma → coller l'URL du fichier local ou utiliser l'extension Chrome
4
Tous les écrans s'importent en frames Figma éditables
